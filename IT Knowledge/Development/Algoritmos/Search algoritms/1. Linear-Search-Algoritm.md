Se revisa cada elemento de una lista.

```typescript
function linearSearch(arr: number[], target: number): number {
    for (let i = 0; i < arr.length; i++) {
        if (arr[i] === target) {
            return i; // Elemento encontrado
        }
    }
    return -1; // Elemento no encontrado
}
```
