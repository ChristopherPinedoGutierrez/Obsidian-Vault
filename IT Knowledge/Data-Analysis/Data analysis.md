# Descripcion

Organizar los datos provenientes de diversos canales para lograr comunicar efectivamente.

# Categorias

- Descriptivo
	- Que ha sucedido. ( KPI's, ROI)
- Diagnóstico
	- Porque ha sucedido. (Se usa los descriptivos como base)
- Predictivo
	- Que ocurrira.
- Prescriptivo
	- Que hacer para logr un objetivo.
- Cognitivo
	- Obtiene conclusiones en funcion a datos y patrones existentes.

# Roles en los datos

- Analista de negocios
	- Esta mas cerca y mas al tanto del modelo de negocio y la empresa siendo especialista en la interpretación de datos.
- Analista de datos
	- Maximiza el valor de los datos, gestionando, limpiando y transformando datos.
	- Trabaja con partes interesadas para identificar requisitos de datos y transformalos en reportes relevantes.
	- Administra recursos de Power BI
	- Esta en cnotacto tambien con perfiles tecnicos como ingenieros de datos y admins de bd para garantizar el acceso a los datos e incluso mejorar procesos 
- Ingeniero de datos
	- Administran configuran y protegen el flujo de datos estructurados y no estructurados del negocio en local y en la nube. 
	- Aglutinan los datos (limpian y transforman.) acelerando el desarrollo de proyectos.
- Científico de datos
	- Realizan analisis avanzados para extraer valor.
	- Usan deep learning en problemas complicados usando algoritmos personalizados complejos.
	- Trabajan con los analistas dandole a estos los datos para que creen informes
- Administrador de base de datos DBA
	- Administra e  implementa aspectos operativos e las soluciones de lpataformas de datos web y local como azure y sqlserver.
	- Administra el estado general de la bd y el hardware donde se encuentra.

# Preparacion de un analista de datos

- Preparacion
	- Convertir datos sin procesar en informacion comprensible
	- Garantizar la integridad de datos.
	- Entender como obtener los datos
- Modelado
	- Determinar la relaciones para terminar con el modelo semantico
- Visuaización
	- Creacion de informes que soluciones los problemas que se esperan.
- Analisis
	- Interpretar informacion.
	- Detectar patrones, tendencias y predecir resultados.
- Administración
	- Administrar recursos y supervisar el uso
